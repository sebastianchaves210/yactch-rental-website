Custom Animation

In this template, we are using GSAP and Lenis. The code for these animations are located in the Page Scroll Component. Here is how you can modify or remove the custom animation.

1. Navigate to any page using webflow designer
2. Look for Global Scripts component in the Nabvigator
3. Double click on Global Scripts Component or right click and then click on edit component from the context menu to edit the component
4. Once you are on the edit component screen, look for Code Embed Settings on the right sidebar
5. Click on the open code editor button just under the Code Embed Settings
6. Change any values in the custom code and save to apply the changes
